Kurslarimiz haqqinda

Kurslarimizda siz 6 ay dawaminda Frontend web programmalastiriwdi toliq ha'm sapali o'zlestirip shig'asiz. Sizge en' aqirg'i ha'm zamanago'y texnologiyalar , ha'r bir programmist biliwi kerek bolg'an bilimler , komandada islewde qollaniliwshi a'sbaplar ( Git ha'm GitHub ) , trenddegi programmalastiriw tili u'yretiledi . Tek g'ana teoriya emes , ha'r ku'ngi u'yrengen bilimlerin'izdi a'melde de qollap barasiz . Sonday-aq kurs dawaminda siz keminde 2 real proekt islep shig'asiz . Bul degeni erten' portfolin'izda 2 proekt boladi degen so'z . Kurslarimiz juwmag'inda jaqsi o'zlestirgen oqiwshilar o'z firmamizg'a ( qa'lewi boyinsha ) jumisqa qabillanip , joqari da'ramatli ayliqqa erisedi . Yamasa Qaraqalpaqstanda ashiliwina tez ku'nler qalg'an IT Parkta islep qaliw imkaniyatina iye boladi . Basqa pitiriwshiler de o'z qa'lewi boyinsha stajirovka o'tewi , ha'm jumis islewi mu'mkin .

Sizge neler u'yretiledi ?

HTML

HTML - (Hypertext markup language) ha'r qanday veb sayttin' en' tiykarg'i bo'legi bolip , onda barliq veb saytlardin' qa'lipleri jaratiladi dewge boladi .

CSS

CSS - (Cascading style sheet) veb saytlardin' stilleri , ren'leri , ha'r qiyli animaciyalar ha'm effektler , transformaciyalar ha'm basqa da ko'plegen elementlerge suliwliq beriw ushin qollaniladi .

SASS

SASS - (Syntactically awesome style sheets) bul CSS preprocessori bolip, onda CSS arqali stil beriledi . Biraq ayirmashilig'i sintaktik jaqtan qolaylig'inda . Ol CSSte stil jaziwdi an'sat ha'm a'piwayilastiradi .

Bootstrap 5

Bootstrap - ko'plegen qolayliqlarg'a iye HTML ha'm CSS kitapxanasi . Ol arqali saytlarimizdag'i kartochkalar , knopkalar , formalar ha'm ko'plegen basqa elementler an'sat jaratiladi .

JavaScript

JavaScript - bul ha'zirgi da'wirdegi en' taniqli ha'm ko'pler paydalanatug'in programmalastiriw tili . JavaScript arqali siz veb saytlar jaratiw menen birgelikte , kompyuterler ha'm mobil qurilmalarg'a programma du'zesiz , sonday-aq ha'tteki robotlarg'a da programma jaza alasiz .

Git ha'm GitHub

Git ha'm GitHub - arqali siz bir proekt u'stinde bir neshe programmist bolip birgelikte islewin'iz mu'mkin . Sonday aq kodlarin'izdi global ta'rizde ja'riyalawdi qa'lesen'iz de GitHub sizge u'lken ja'rdem beredi .

NodeJS

NodeJS - bolsa JavaScriptti brauzerden sirtqi ta'repte isletip beriwshi ortaliq bolip , Chrome brauzerinin' V8 dvijogin C++ tiline qoyiw arqali islengen . NodeJS JavaScripttin' imkaniyatlarin min'lap ese asirdi desek asira aytqan bolmaymiz .

React

React - bul Facebook kompaniyasi ta'repinen jaratilg'an ha'm qollap quwatlanatug'in JavaScript kitapxanasi bolip , paydalaniwshi interfeysin jaratiwda qollaniladi . Ha'zirde ha'r qanday frontend programmist React , Angular yamasa VueJSti biliwi talap etiledi . Frontend kursimizdin' en' tiykarg'i logikasi da usi Reactqa qaratilg'an boladi .
